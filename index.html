Image Search Abstraction Layer

Usage : " https://mgjean-img-search.glitch.me/query&count&offset "

  • query : the thing you want to search
  • count : number of results
  • offset : skip number of result from the start

Search : " https://mgjean-img-search.glitch.me/cats&5&0 "

Return : '{"name" : "...", "content" : "...", "thumbnail" : "...", "source" : "...", "published" : "..."}'

Usage : " https://mgjean-img-search.glitch.me/history/limit "

  • history : view search history
  • limit : number of search history to display

Searched : " https://mgjean-img-search.glitch.me/history/10 "

Return : '{"query" : "...", "time" : "..."}'
